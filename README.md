# JavaScript_TypeScript_Learning

This repository contains beginner-friendly JavaScript and TypeScript learning examples.

## Folder structure

- `01_Javascript/` – JavaScript fundamentals organized by topic
- `02_Typescript/` – TypeScript and Playwright automation learning content

## How to use

- Open the folder in your editor
- Run JavaScript examples with Node.js: `node <filename>.js`
- Review README files inside each folder for topic details
