# 01_Javascript

This folder contains JavaScript examples for foundational language concepts.

## Subfolders

- `1_Basics/` – core syntax, operators, variables, and expression behavior
- `2_Conditional_Statements/` – `if`, `else`, and `switch` statements
- `3_UserInput/` – basic console user input examples using Node.js
- `4_Loops/` – `for`, `while`, and `do...while` loops
- `5_Arrays/` – array syntax and operations
